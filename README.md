# Lucas Babetto Atelier — Website

**Site profissional de alta costura para o Lucas Babetto Atelier, Maringá – PR**

---

## 🌟 Sobre o Projeto

Site institucional de nível Squarespace para o **Lucas Babetto Atelier** — atelier de moda feminina especializado em vestidos exclusivos sob medida em Maringá, Paraná. O designer Lucas Babetto cria peças únicas com bordado artesanal à mão para momentos especiais.

---

## ✅ Funcionalidades Implementadas

### 🎬 Intro / Splash Screen
- Tela de introdução elegante com animação de partículas douradas
- Tipografia luxuosa com nome "Lucas Babetto Atelier"
- Música gerada via **Web Audio API** (acordes arpejados estilo harpa/piano) — toca no primeiro toque/clique do usuário
- Transição suave de fade+scale para o site principal
- Auto-dismiss em 4.5 segundos OU ao clicar em "Entrar"

### 🎥 Video Background Hero
- Vídeo de fundo em loop infinito, mudo, sem controles
- Rotação automática entre múltiplos vídeos a cada 15 segundos com fade
- Camada transparente de proteção que impede qualquer clique/interação com o vídeo
- Overlay escuro (45%) para legibilidade do texto
- Suporte mobile com poster fallback

### 🧭 Navegação
- Navbar fixa com efeito glassmorphism ao scroll
- Menu hambúrguer responsivo para mobile
- Links com animação de underline dourado
- CTA "Agendar Horário" em destaque
- Ícones de Instagram e WhatsApp

### 📖 Seções
1. **Hero** — Vídeo background + nome do atelier + CTAs + stats
2. **Sobre** — Apresentação do atelier com imagens empilhadas e pilares
3. **Serviços** — 6 cards de serviços com ícones e links para WhatsApp
4. **Galeria** — Grid masonry com imagens reais do Instagram + lightbox
5. **Showcase de Vídeos** — 3 mini-vídeos do processo criativo
6. **Agendamento** — Formulário completo de contato
7. **Contato** — Informações de contato + mapa do Google Maps
8. **Rodapé** — Links, redes sociais, direitos autorais

### 📝 Formulário de Agendamento
- Campos: Nome completo, E-mail, WhatsApp, Tipo de ocasião, Mensagem
- Envio via **FormSubmit.co** para `Atelier.lucasbabetto@gmail.com`
- Auto-resposta por e-mail para o cliente
- Máscara automática no campo de telefone
- Feedback visual de sucesso após envio

### 💬 WhatsApp Flutuante
- Botão fixo bottom-right com animação de pulso
- Tooltip ao hover: "Fale conosco"
- Link direto para o WhatsApp do atelier
