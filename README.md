# Lucas Babetto Atelier — Website Oficial

## Visão Geral
Site premium para o **Lucas Babetto Atelier**, com estética editorial de luxo inspirada nos melhores templates Squarespace. Desenvolvido com HTML, CSS e JavaScript puros, oferecendo uma experiência visual sofisticada, fluida e responsiva.

## ✅ Funcionalidades Implementadas

### Design & UX
- **Cursor personalizado** com ponto dourado e anel animado com efeito de follow
- **Page loader** animado com as iniciais LBA
- **Navbar** fixa e transparente que muda de estilo ao scrollar
- **Menu mobile** fullscreen com links animados
- **Animações de entrada** com Intersection Observer (scroll reveal)
- **Parallax** na imagem hero
- **Tilt effect** nos cards de depoimentos
- **Marquee** animado com pausa ao hover

### Seções
1. **Hero** — Imagem full-screen com título tipográfico grande, overlay, badge rotativo e scroll indicator
2. **Marquee Strip** — Faixa dourada com texto animado (Sob Medida, Alfaiataria de Luxo, etc.)
3. **Sobre / Atelier** — Grid com imagem + frame decorativo, estatísticas animadas e assinatura
4. **Coleção** — Grid editorial assimétrico com hover overlay e CTA para WhatsApp
5. **Processo** — Seção escura com imagem de fundo, 5 etapas do atelier
6. **Galeria** — Layout masonry com hover reveal
7. **Depoimentos** — 3 cards com card central em destaque + tilt effect
8. **Contato** — CTA fullscreen escuro com botão WhatsApp principal
9. **Footer** — 3 colunas com branding, navegação e contato
